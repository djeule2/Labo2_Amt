# Labo2_Amt

Partie JPA
- pour l'execution on pourra reccuperer le fichier e-commerce.war et le déployer depuis un serveur d'application .
- on peut également lancer l'application depuis un ide java.


Partie JMeter
- il suffira installer le zip comportant l'executable de jmeter dans sa machine  puis lancer cette exécutable. Ensuite vous pourrez importer le fichier <<Virtual users.jmx>>
les configurations sont intégrer dans se fichier. 
par ailleurs pour l'exécution il faut au préable que la partie jPA du e-commerce soit en marche

partie Semelium
- vous déposez l'executable de chromeDriver dans un repertoire C:\\chrommedriver\

Puis vous pouvez lancer application depuit intellij
